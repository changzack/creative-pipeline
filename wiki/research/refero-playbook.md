# Refero Research Playbook

Proven queries and strategies for the Refero MCP research tool. Updated after each run.

## Query Strategy

### Multi-angle approach (run 10-15 queries)
1. **Direct match** — the exact product/format described in the brief
2. **Visual technique** — specific CSS/WebGL techniques mentioned in brief
3. **Adjacent category** — related but different content types for unexpected inspiration
4. **Layout/composition** — structural patterns regardless of content
5. **Experimental** — creative/generative/unusual interpretations

### Queries to Avoid
- Too broad: "dark mode design", "mobile app" — returns too many generic results
- Too specific: exact brand names return nothing
- Content-matching: "sneaker ranking" — returns sneaker sites, not design inspiration

## Moodboard Composition Rules

### Must have diversity across 3+ categories:
- Data visualization / infographic
- Editorial / magazine layout
- Product / e-commerce
- Social / share card
- Dashboard / dark UI
- Experimental / art-directed

### Quality gates:
- Real product screens, NOT aggregator listing pages
- High enough resolution to see typography and color details
- 8-12 images total
- Each image should contribute a DIFFERENT visual idea

### Lesson: Moodboard content drift (V3b)
The moodboard's visual content can overwhelm the brief's text instructions. If moodboard shows Spotify Wrapped, builders build Spotify Wrapped. Moodboard must be STYLE references, not CONTENT references.
